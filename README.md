# ☕ Café Aroma — Digital Experience
A refined web project that translates the cozy atmosphere of a physical café into a modern digital interface. This platform showcases the fusion of minimalist design with robust frontend logic.
## 📄 Pages Included
 * 🏠 **Home Page** (High-impact hero section with cinematic design)
 * 📖 **Menu Explorer** (Interactive categories for drinks and snacks)
 * 🖼️ **Visual Gallery** (A curated grid of our café's atmosphere)
 * 🌿 **Our Story** (Origins, sustainability, and team values)
 * ✉️ **Contact Hub** (Validated form and real-time location info)
## ⚙️ Features
 * 🌐 **Multi-page Architecture**
 * 🎨 **Premium Minimalist UI** (Using Playfair Display & Poppins)
 * 🔗 **Dynamic Navigation** (Smooth scroll and interactive header)
 * 📱 **Mobile Navigation Toggle** (Adaptive drawer menu for smaller screens)
 * ⚡ **JS Interaction** (Real-time menu filtering and input validation)
 * ☕ **Coffee-Centric UX** (Designed for a cozy user experience)
